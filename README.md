AI Grand Prix – Autonomous Racing Drone
Overview

This project is part of the AI Grand Prix competition that is hosted by Anduril Industries , where the goal is to design and develop an autonomous drone capable of navigating a race track without human control. The drone uses computer vision, sensors, and intelligent algorithms to detect gates, avoid obstacles, and complete the track as efficiently as possible.
The focus of this project is to experiment, learn, and build a working autonomous system by combining hardware and software.

the Goal is to 
Build an autonomous drone that can navigate a racing track
Use AI and computer vision to detect gates and track paths
Integrate sensors and control systems for stable flight
Test and improve the system through simulation and real-world experiments

_Technologies_Used_

ROS/ROS2

Gazebo (simulation)

RViz (visualization)

Python

Computer Vision (OpenCV / AI models)

Flight controller integration
